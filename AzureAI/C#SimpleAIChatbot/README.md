# ASP.NET Azure Foundry AI Chatbot
A sample ASP.NET chatbot that leverages Azure Foundry AI models to respond to user requests.

## Quick Start

### 1. Configure your API key
```bash
cp .env.example .env
```
Open .env and replace the placeholders with your real Azure AI API keys

### 2. Run the application
The easiest approach is to run the web application using Visual Studio.
