External launcher

The widget uses launcher="external", so the default floating bubble is hidden. Wire your own buttons to .open(), .close(), or .toggle() on the custom element.

To post a first message automatically (no typing), call .open('your question') or .open({ prompt: '…' }), or set initial-prompt="…" so each time the panel opens that text is sent once.

Example table: hotel_reservations, id column: reservation_id.

Omit launcher to show the built-in launcher. Rebuild the bundle after frontend changes: npm run build in AIAssistant/frontend.
